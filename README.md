# Quadcopter-Design-Using-Inventor
This repository contains the documentation and design files for the Quadcopter Design project completed for MTE 361 at the American University of Science & Technology, Department of Mechatronics Engineering.
## Overview
The project focuses on designing a stable and maneuverable quadcopter using Autodesk Inventor. It covers the entire mechanical design process, including CAD modeling, simulation, and stress analysis, aimed at achieving optimal performance and payload capacity.
## Key Features
* Stable and Maneuverable: Designed to ensure stability and responsiveness during flight.
* Payload Capacity: Capable of carrying payloads as part of its design considerations.
* CAD Modeling: Detailed 3D modeling of all components using Autodesk Inventor.
* Simulation and Analysis: Includes stress analysis to validate structural integrity under load conditions.
* Comprehensive Documentation: Provides a thorough overview of the design process, materials selection, and testing results.

## Objectives
* Mechanical Design: Provide a comprehensive overview of the mechanical design process using Autodesk Inventor.
* CAD Modeling and Simulation: Detail the stages of CAD modeling, simulation, and stress analysis.
* Design Considerations: Discuss material selection, motor and propeller choices, and weight distribution
* Prototype Testing: Present results of testing to validate stability, responsiveness, and payload capacity.
* Educational Resource: Serve as a reference for designing quadcopters using CAD software.

## Tools and Components Used
* CAD Software: Autodesk Inventor for 3D modeling and simulation.
* Finite Element Analysis (FEA): Used for stress analysis to ensure structural integrity.
* Components: Motors, propellers, PC/ABS plastic for frame, and other necessary hardware.
* Testing Equipment: Setup for flight testing and performance evaluation.

## Conclusion
The Quadcopter Design project demonstrates the application of mechanical engineering principles to create a functional and efficient quadcopter design. It emphasizes the use of CAD software for detailed modeling and simulation to optimize performance and reliability. This repository serves as a valuable resource for students and engineers interested in UAV design and mechanical engineering.
